Architectural.ly

Architectural.ly is an application that was built for architecture enthusiasts, and adventurers. This app allows adventurers to share and save their favorite neighborhood architecture locations on a map by simply adding a new spot. The app detects the user's location, the user uploads a picture, gives their spot a brief description and categorizes the type of house and a marker will be placed on the map. Now enthusiasts, tourists and travellers can build out self guided architecture tours!

  • Technologies Used: Javascript, jQuery, Google Maps API, Nodejs, MongoDB, Amazon AWS S3 for application data storage, Heroku for web application hosting, Handlebars as the templating engine, HTML5 Geolocation, Materialize CSS Framework, CSS
  • Date: January 2017
